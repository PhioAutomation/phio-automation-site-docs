# Boolean

### Description

The `BOOL` is the workhorse of PLC programming. A boolean variable has only two states `FALSE` and `TRUE`.

### Declaration

```iecst
a : BOOL;           // Initializes to FALSE
b : BOOL := TRUE;   // Initialized to TRUE

a := TRUE;
a := FALSE;
```
