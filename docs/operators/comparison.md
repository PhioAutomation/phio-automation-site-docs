# Comparison Operators

```iecst
a : INT := 6;
b : INT := 3;

print(a < b);       // False
print(a <= b);      // False
print(a > b);       // True
print(a >= b);      // True
```

```iecst
a : INT := 6;
b : INT := 3;

print(a == b);      // False
print(a <> b);      // True
```
