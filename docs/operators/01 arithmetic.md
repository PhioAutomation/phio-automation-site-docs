# Arithmetic

```iecst
a : INT := 6;
b : INT := 3;

print(a + b);   // 9
print(a - b);   // 3
print(a * b);   // 18
print(a / b);   // 2
```

```iecst
a : INT := 6;
b : INT := 3;

print(a ** b);  // 216
```
